# mubeen.dev

Personal portfolio. Built with Next.js, TypeScript, and Tailwind CSS. Designed around a distributed systems trace metaphor.

## Stack

- **Framework:** Next.js 14+ (static export)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animation:** Framer Motion
- **Fonts:** JetBrains Mono + Inter
- **Hosting:** Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```

## Adding a Project

Open `src/content/projects.ts` and append a new project object. No other files need to change. See the existing entries for the shape.

## Deploying

Pushes to `main` deploy automatically via Vercel.
